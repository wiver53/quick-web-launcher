# 快捷启动器

一个轻量、高效的网页/应用快速启动工具，支持一键打开网页、唤起安卓应用，同时提供个性化设置功能。


## 🚀 功能特点
1. **网页极速打开**  
   输入网址直接跳转，输错内容会自动触发搜索（如输入“百度”自动跳转到搜索结果页）

2. **常用网页/应用集合**  
   - 内置常用网页快捷入口，像浏览器标签页一样随点随用
   - 支持安卓应用包名唤起（需设备权限支持）

3. **个性化设置**  
   - 动画控制：开启/关闭动画、调整动画速度（兼顾强迫症与丝滑体验）
   - 性能优化：关闭图标功能，弱网环境下访问更流畅
   - 字体适配：自由调整字体大小，适配不同设备/阅读习惯
