## 📖 Introduction | 项目介绍
### English
This is a clean, ad-free, single-file HTML tool designed to solve the problem that the top search bar of some mobile browsers is inconvenient and unresponsive. It supports quick access to web pages and experimental app launching, and is hosted on Cloudflare Pages through GitHub open source.

### 中文
这是一款简洁无广告的单文件HTML工具，专为解决部分移动端浏览器顶部搜索栏难用、响应迟缓的问题打造。支持快速访问网页、实验性应用唤起，通过GitHub开源并托管于Cloudflare Pages。

## ✨ Features | 核心功能
### English
- Fast webpage opening: Stable and usable on all browsers & devices
- Experimental app launch: Support opening apps by package name (limited by browser policies)
- No installation, no ads, no tracking, no permissions required
- Can be set as browser homepage to replace the default search bar

### 中文
- 快速打开网页：全浏览器、全设备稳定可用
- 实验性应用唤起：支持包名打开应用（受浏览器策略限制）
- 无需安装、无广告、无数据追踪、无需任何权限
- 可设置为浏览器主页，替代原生搜索栏

## 🚀 Usage Guide | 使用教程
### English
1. Open the online link in any mobile browser
2. Enter the URL in the input box and click "Open"
3. Set this page as your browser homepage for faster access
4. The app launch function is experimental and usually unavailable

### 中文
1. 在任意移动端浏览器打开在线链接
2. 在输入框输入网址，点击打开即可跳转
3. 可将本页面设置为浏览器主页，提升访问速度
4. 应用打开功能为实验性功能，通常无法使用

## ⚠️ Known Issues & Bugs | 已知问题
### English
- Webpage access: Fully functional and compatible
- App launch: Generally unavailable
  Modern browsers restrict web pages from opening third-party apps for security reasons, which is a system-level restriction, not a code bug.

### 中文
- 网页访问功能：完全正常，全兼容
- 应用唤起功能：普遍无法打开
  现代浏览器出于安全策略，限制网页打开第三方应用，属于系统层面限制，并非代码BUG。
